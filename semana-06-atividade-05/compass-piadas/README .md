
# Compass Piadas

```md

Este é o projeto **Compass Piadas**, que permite carregar piadas de um arquivo JSON, adicionar novas piadas em tempo real, deletar piadas, e mostrar as respostas ao clicar em um botão **Ver resposta**. O projeto foi construído utilizando **React** e js**.

## Pré-requisitos

- Node.js (versão >= 16.x.x)
- NPM (versão >= 8.x.x)

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado na sua máquina.

## Como clonar o projeto

1. Abra o terminal.
2. Navegue até o diretório onde deseja salvar o projeto.
3. Execute o seguinte comando para clonar o repositório do GitHub:

```bash
git clone https://github.com/Alexverissimo/alex-verissimo-estudos/semana-06-atividade-05
```

4. Acesse a pasta do projeto:

```bash
cd compass-piadas
```

## Como instalar o React com Vite

Caso ainda não tenha configurado o **React** no projeto, siga os passos abaixo para configurar utilizando o **Vite**:

1. No terminal, dentro do diretório do projeto, execute o seguinte comando para criar o projeto com Vite:

```bash
npm create vite@latest
```

2. Quando solicitado, escolha o nome do projeto (ou pressione enter para manter o padrão).
3. Selecione a opção `React` como framework.
4. Selecione `JavaScript` ou `TypeScript` conforme preferir.

Após a criação do projeto, entre na pasta do mesmo com:

```bash
cd nome-do-seu-projeto
```

## Instalar as dependências

Depois de configurar o projeto com Vite, você precisa instalar as dependências do `node_modules`. Para isso:

1. No terminal, dentro do diretório do projeto, execute o comando:

```bash
npm install
```

Esse comando vai instalar todas as dependências listadas no arquivo `package.json`.

## Rodando o projeto

Com as dependências instaladas, agora você pode rodar o projeto. Para isso, execute o seguinte comando:

```bash
npm run dev
```

Isso iniciará o servidor de desenvolvimento, e o projeto estará disponível no navegador.

### Acesse o projeto no navegador:

O servidor rodará normalmente na seguinte URL: [http://localhost:](http://localhost:).

Agora você pode acessar a aplicação localmente e ver as funcionalidades de piadas em ação.

---

## Tecnologias utilizadas

- React
- Node.js
- Vite
